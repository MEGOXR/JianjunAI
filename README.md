# JianjunAI Project

This is a full-stack application with a WeChat Mini Program frontend and Node.js backend.

## Project Structure

```
JianjunAI/
├── frontend/      # WeChat Mini Program frontend
├── backend/       # Node.js backend server
└── README.md      # This file
```

## Setup

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
Open the `frontend` folder in WeChat Developer Tools.

## Technologies Used
- Frontend: WeChat Mini Program
- Backend: Node.js, Express
- AI Integration: OpenAI API
